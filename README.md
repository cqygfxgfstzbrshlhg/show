# 课程作业展示页

这是一个基于 **HTML + CSS** 的静态网页课程作业项目，用于展示个人基本信息、作业说明、学习过程与成果。

## 项目结构

- `index.html`：页面结构与内容
- `style.css`：页面样式与响应式布局

## 功能说明

- 课程作业展示首页（Hero 区）
- 基本信息展示
- 作业说明与任务清单
- 学习过程时间线
- 成果展示区
- 联系方式区
- 移动端适配（响应式）

## 本地预览

直接双击打开 `index.html`，或使用任意静态服务器预览。

## GitHub Pages 部署

1. 将项目推送到 GitHub 仓库
2. 进入仓库 `Settings -> Pages`
3. 在 `Build and deployment` 中选择：
   - Source: `Deploy from a branch`
   - Branch: `main` / `/root`
4. 保存后等待部署完成，即可获得公开访问链接

## 备注

本项目不依赖前端框架或构建工具，适合作为入门级静态网页课程作业提交。
